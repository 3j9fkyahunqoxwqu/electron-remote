    RecursiveProxyHandler

    RecursiveProxyHandler is a ES6 Proxy Handler object that intercepts method invocations and returns the full object that was invoked. So this means, if you get a proxy, then execute foo.bar.bamf(5), you'll recieve a callback with the parameters "foo.bar.bamf" as a string, and [5].
